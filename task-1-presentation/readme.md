# Task 1: Digital Literacy Awareness Infographic

## Overview
This task focuses on creating a one-page visual infographic to spread awareness about digital literacy among students. The goal is to present important concepts in a simple, clear, and visually engaging way.

---

## Tool Used
The infographic was created using Canva, a user-friendly design tool that allows easy creation of visual content using templates, icons, and layouts.

---

## Topics Covered
The infographic includes the following key aspects of digital literacy:

- **What is Digital Literacy**  
  Basic understanding of using digital tools effectively and responsibly  

- **Safe Internet Practices**  
  Tips like protecting personal information and avoiding suspicious links  

- **Professional Online Presence**  
  Importance of maintaining a positive digital identity on platforms like LinkedIn and GitHub  

---

## Work Completed
- Designed a single-page infographic using a digital design tool  
- Included key concepts in a simple and visually appealing format  
- Focused on clarity, readability, and relevance for students
